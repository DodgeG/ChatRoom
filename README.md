# ChatRoom

## 项目简介

ChatRoom 是一个基于 Java Socket 的课程聊天室项目，目标是通过一个可运行的多人聊天系统，实践网络编程中的连接管理、并发处理、协议设计与消息广播等核心能力。

项目支持多客户端同时在线，能够处理用户加入、聊天消息转发和退出通知。仓库同时提供源码、可执行 JAR 和实验报告，方便快速演示与复现实验结果。

## 项目架构

系统采用经典 C/S（Client/Server）架构：

- Client
  - 负责用户输入、消息发送与消息接收展示
  - 使用独立发送线程处理控制台输入
- Server
  - 监听端口，接收新连接
  - 为每个客户端创建独立处理线程
  - 将某个客户端消息广播给其他在线客户端

消息处理流程：

1. 客户端连接服务端并发送“加入聊天室”消息
2. 服务端接收后广播加入提示
3. 客户端发送普通消息，服务端向在线用户广播
4. 客户端发送 `exit`，服务端广播退出并回收连接

## 功能特性

- 服务端监听端口并为每个客户端创建独立线程
- 客户端可输入昵称并加入聊天室
- 支持群聊消息广播
- 支持 `exit` 指令退出聊天室
- 实现了简单消息编码（字符偏移 + 结束符）用于传输

## 项目结构

- `src/Server.java`: 服务端入口与客户端会话处理线程
- `src/Client.java`: 客户端入口、发送线程与接收逻辑
- `Server.jar` / `Client.jar`: 已构建可执行包
- `report.md` / `report.pdf`: 实验报告与截图

## 运行方式

### 方式 1：直接运行 JAR

```bash
java -jar Server.jar
java -jar Client.jar
```

### 方式 2：从源码编译运行

```bash
javac -d bin src/*.java
java -cp bin Server
java -cp bin Client
```

## 使用说明

1. 先启动服务端并输入监听端口
2. 再启动一个或多个客户端，输入端口与昵称
3. 在客户端输入消息发送群聊
4. 输入 `exit` 退出聊天室

## 协议备注

- 客户端首条加入消息格式：`<name>&)start`
- 普通聊天消息格式：`<name>&<message>`
- 传输层使用字符偏移（ASCII + 13）并以 `2333` 作为结束符
